# Characters Microservice (GAMEDEV)

**Autor / Author:** Andrés Pestana Hidalgo

Este microservicio gestiona **personajes** y sus **estadísticas base**.

---

## ES — Resumen

### Tecnologías
- Java 17
- Spring Boot
- Maven
- H2 (base de datos en memoria)

### Puerto
- **8080** → `http://localhost:8080`

### Ejecución (Windows)

**PowerShell**
```powershell
cd C:\dev\gamedev\characters\characters
mvn spring-boot:run
```

**CMD**
```bat
cd /d C:\dev\gamedev\characters\characters && mvn spring-boot:run
```
